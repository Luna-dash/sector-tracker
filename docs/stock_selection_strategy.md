# 三阶段量价选股策略

## 整体结构

每个阶段有**硬规则**（不过就淘汰）和**打分排序**（过关后排名用）。
三个阶段从当前交易日往回推，顺序为：突破 → 回踩 → 确认。

核心参数：EMA13、EMA55。

---

## 第一阶段：突破

### 回溯窗口
15个交易日（从今天往前数第1天到第15天）

### 120日最高价计算窗口
从第16天到第135天（即15天窗口之前再推120天），取该区间最高价。

### 硬规则
1. 15天内至少有一天收盘价 > 120日最高价
2. 突破日 = 首次超过120日最高价的那天
3. 打分 ≥ 10分

### 打分
- 基准：50分
- 最佳位置：第11天（从今天往前数）
- 偏离扣分：
  - 往前偏离1天（更早）：-8分/天
  - 往后偏离1天（更近）：-5分/天
- 低于10分：否决
- 有效窗口：约第6天 ~ 第13天

**计算公式**：
```
breakout_score = 50 - 8 × max(0, 11 - breakout_day) - 5 × max(0, breakout_day - 11)
其中 breakout_day 为突破日距今天的天数（1=今天，15=15天前）
```

---

## 第二阶段：回踩

### 回溯窗口
7个交易日（从今天往前数第1天到第7天）

### 硬规则
1. 7天内的最低价落在 EMA13 附近：
   - 上限：EMA13 × 1.02（高于EMA13 2%）
   - 下限：EMA13 × 0.97（低于EMA13 3%）
2. 回踩日收盘价 ≥ EMA13 - 0.5%
3. 7天内收盘价任何时候都不能跌破 EMA55

### 回踩日确定
取以下两个日期中较早的：
1. 7天内最低价最低的那天
2. 首次最低价低于 EMA13 的那天

### 打分
- 基准：30分
- 最佳位置：第5天（从今天往前数）
- 偏离扣分：
  - 往前偏离1天（更早）：-5分/天
  - 往后偏离1天（更近）：-3分/天
- 回踩日最低价偏离 EMA13 的惩罚：
  - 低于 EMA13：每1%扣3分
  - 高于 EMA13：每1%扣2分
- 回踩日收盘价回升奖励：
  - 收盘价相对最低价每高1%：+2分
- 最终分数不能低于0

**计算公式**：
```
pullback_score = 30
  - 5 × max(0, 5 - pullback_day)    # 往前偏离
  - 3 × max(0, pullback_day - 5)     # 往后偏离
  - 3 × max(0, (EMA13 - low) / EMA13 × 100)   # 低于EMA13的惩罚
  - 2 × max(0, (low - EMA13) / EMA13 × 100)   # 高于EMA13的惩罚
  + 2 × (close - low) / low × 100    # 收盘价回升奖励
pullback_score = max(0, pullback_score)
其中 pullback_day 为回踩日距今天的天数
```

---

## 第三阶段：确认

### 回溯窗口
5个交易日（从今天往前数第1天到第5天）

### 硬规则
1. 最多只有1天的最低价可以低于 EMA13
2. 所有收盘价 ≥ EMA13
3. 最近收盘价 ≤ EMA13 × 1.11（涨幅不超过11%）

### 打分（基准20分）
- 每有1天最低价低于 EMA13：-5分
- 每有1天最低价低于回踩日最低价：-5分
- 以上两项可叠加（某天既低于EMA13又低于回踩日最低价：-10分）
- 每有1天收盘价和成交量都高于前一日：+3分
- 最终分数不能低于0

**计算公式**：
```
confirm_score = 20
  - 5 × days_low_below_ema13          # 低于EMA13的天数
  - 5 × days_low_below_pullback_low    # 低于回踩日最低价的天数
  + 3 × days_price_volume_up           # 价量齐升的天数
confirm_score = max(0, confirm_score)
```

---

## 输出分类

### A类（回踩到位）
- 突破硬规则通过
- 回踩硬规则通过
- 回踩打分 ≥ 0
- 排序依据：突破打分 + 回踩打分

### B类（确认完成）
- 突破硬规则通过
- 回踩硬规则通过
- 确认硬规则通过
- 回踩打分 ≥ 0
- 确认打分 ≥ 0
- 排序依据：突破打分 + 回踩打分 + 确认打分

---

## 说明

- 所有回溯窗口均从当前交易日往前推
- 突破、回踩、确认三个窗口允许重叠（自然结果，不额外约束）
- "跌破"均指收盘价（真实跌破）
- "最低价"指当日K线最低价
