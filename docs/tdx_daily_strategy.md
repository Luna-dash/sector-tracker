# 通达信日线策略整合说明

## 结论

当前已把你给的几套公式整理成一套“日线收盘后选股”逻辑，并接入现有选股流程。  
盘中逻辑和需要额外数据源的部分先不接，后面再扩。

## 这次纳入的核心逻辑

### 1. RPS 多周期强度

来源：
- `RPS10 / RPS20 / RPS50 / RPS120 / RPS250`
- `三周期红`
- `四周期红`

当前实现：
- 使用板块内候选股票的阶段涨幅，近似计算 `RPS10/20/50/120/250`
- 阈值默认 `> 90`
- `三周期红`：
  - `RPS10 > 90`
  - `(RPS20 > 90 或 RPS50 > 90)`
  - `(RPS120 > 90 或 RPS250 > 90)`
- `四周期红`：
  - `RPS10 > 90`
  - 且 `RPS20 / RPS50 / RPS120 / RPS250` 里至少 3 个大于 90

接入方式：
- 三周期红加分
- 四周期红加更高分
- 报告中输出 RPS 摘要

说明：
- 通达信原公式依赖外部 RPS 数据，这里暂时做成“板块内相对强度近似版”

### 2. 学友大单的日线代理

来源：
- `DDX`
- `DDX1`
- `DDX2`
- `预警: DDX>0 AND MA(V,5)>MA(MA(V,5),5) AND CROSS(DDX1,DDX2)`

当前实现：
- 用 K 线位置关系估算主动买卖量差
- 用换手率近似归一化大单强弱
- 计算 `DDX`、`DDX1`、`DDX2`
- 满足以下条件时记为 `资金预警`
  - `DDX > 0`
  - `5 日均量 > 5 日均量的再均线`
  - `DDX1` 上穿 `DDX2`

接入方式：
- 作为资金确认信号加分

说明：
- `LARGEINTRDVOL / LARGEOUTTRDVOL / L2_AMO` 这类 Level-2 数据当前数据链路没有，先不做精确版

### 3. MACD 钝化打板

来源：
- `HDH`
- `ZT`
- `QSDH := HDH AND ZT`

当前实现：
- 计算 `DIF / DEA / MACD`
- 满足以下条件时记为 `MACD点火`
  - MACD 柱在 0 轴上方回升
  - 当日涨幅接近涨停阈值

接入方式：
- 作为点火信号加分
- 当原始信号较弱时，可把 `观察` 提升为 `点火预备`

### 4. 翻身计划

来源：
- `RPS50 > 87`
- 站上年线
- 一个月内曾创 50 日新高
- 最近 30 天站上年线天数在 3 到 29 天之间
- 距离 120 日高点不远

当前实现：
- 识别“刚刚转强，但还不是高位滞涨”的翻身结构

接入方式：
- 命中后加分
- 如果不是回踩买点，信号类型可提升为 `翻身计划`

### 5. 结构突破组合

来源：
- `KD1 / KD2 / KD4 / KD5 / KD6 / KD20`
- `KDZDT2`

当前实现：
- 要求同时满足以下几类条件：
  - 中长周期 RPS 强
  - 当日涨幅至少 5%
  - 结构紧凑，回撤受控
  - 90/100/120 日趋势向上
  - 处于阶段高位附近或创高
  - 成交额明显放大或接近涨停

接入方式：
- 作为最强的突破确认信号加分
- 命中后信号类型直接提升为 `趋势突破`

### 6. 参考止盈线

来源：
- 最近一次大阳线实体中值

当前实现：
- 找最近 10 天内最后一个涨幅大于等于 5% 的交易日
- 用该日实体中值作为 `参考止盈线`

接入方式：
- 仅写入报告和输出结果
- 当前不参与筛选分数

## 暂时不接入的部分

### 1. 盘中逻辑

不纳入本次实现：
- 盘中点火
- 盘中大单跟踪
- 实时图标、实时异动

原因：
- 当前脚本是收盘后分析模型

### 2. Level-2 大单净流入精确版

不纳入本次实现：
- `LARGEINTRDVOL`
- `LARGEOUTTRDVOL`
- `L2_AMO`

原因：
- 当前数据源没有稳定提供这类逐笔/分档数据

### 3. 业绩断层

不纳入本次实现：
- `FINVALUE`
- `FINONE`
- 公告日差、预告断层、快报断层、财报断层

原因：
- 需要额外的财务公告日期与业绩字段数据管道
- 可以在后续单独扩一层“事件驱动”模块

## 当前策略在代码里的落点

- `sector_tracker.py`
  - RPS 多周期强度
  - 大单代理
  - MACD 点火
  - 翻身计划
  - 结构突破
  - 参考止盈线

## 当前使用方式

现有流程不变，策略已经并入选股评分：

```bash
python sector_tracker.py
```

输出结果里会多出：
- `RPS` 摘要
- `策略` 标签
- `参考止盈线`
